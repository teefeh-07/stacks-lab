# Contributing to Stacks Lab

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run tests
5. Submit a pull request

## Commit Convention

We use conventional commits:
- feat: New features
- fix: Bug fixes
- docs: Documentation
- test: Tests
- refactor: Code refactoring

## Code Style

- Use TypeScript
- Follow ESLint rules
- Add tests for new features

## Testing

```bash
npm test
```
